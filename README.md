# Hulu 2.0

## Getting started

Generate your own API key in TMDb and place it as API_KEY on your .env.local

```bash
$ npm install
$ npm run dev
```

## Building and deploying in production

```bash
$ npm run build
$ npm start
```

## Other

```bash
# storybook
$ npm run storybook

# build and export
$ npm run build && next export
```
